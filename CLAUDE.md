# Spine Godot Project

This project uses the Spine-Godot GDExtension (Esoterick Software) for 2D skeletal animation.

## Spine API Rule

Before writing any code that involves Spine nodes, resources, or API calls, always fetch the official docs first:

- Main reference: https://esotericsoftware.com/spine-godot
- Animation/runtime API: https://esotericsoftware.com/spine-runtimes-guide

Do not rely on training data for the Spine-Godot API — it is niche and changes between versions. Always verify node names, property names, and method signatures from the live docs before using them.
